# AI Study Assistant

AI-powered study assistant that converts PDF notes into:

- Flashcards
- Multiple Choice Questions (MCQs)
- Interactive Quizzes
- Quiz Score Reports
- PDF Exports

## Tech Stack

- Python
- Flask
- Google Gemini API
- HTML/CSS
- Render

## Features

✔ PDF Upload
✔ AI Flashcard Generation
✔ AI MCQ Generation
✔ Interactive Quiz
✔ Quiz Scoring
✔ Flashcard PDF Download
✔ Quiz Report PDF Download
